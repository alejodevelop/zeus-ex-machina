# Piskel Sources

Use this folder for editable `.piskel` source files and shared palette files.

## Workflow

- Open `https://www.piskelapp.com/kids/` for the ad-free browser editor the agent can automate most reliably.
- Save source work as `.piskel` files in this tree.
- Export runtime files into `public/assets/sprites/` or `public/assets/tilesets/`.
- Keep filenames in kebab-case and pair important assets with a short brief before drawing.

## Notes

- Prefer local file save and export over Piskel's online gallery.
- Commit the `.piskel` source file whenever the exported art matters to gameplay or polish.
- Use `art/piskel/ASSET-BRIEF-TEMPLATE.md` when you want the agent to create or revise an asset with clear constraints.
