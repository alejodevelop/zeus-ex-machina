# Shared Palettes

These palette files are the stable color references for agent-made pixel art.

## Scaffold Warm

- `#17324a` - deep ink
- `#274864` - harbor blue
- `#405466` - blue gray
- `#6d7d89` - mist
- `#8b5b34` - clay
- `#a85d2d` - rust
- `#c17b45` - amber
- `#f0b46d` - signal gold
- `#dbcab8` - sand
- `#efe6d8` - paper
- `#f8efe3` - linen
- `#fff9f0` - highlight

Use `scaffold-warm.gpl` when the current Piskel session accepts palette import cleanly.
If palette import is awkward, treat this file as the canonical swatch list and enter colors manually.
